## What is spwn?

Claude Code is powerful, but every session starts from scratch and lives and dies
in one terminal window. spwn fixes both. It's a desktop app that organizes your work
into **projects**, lets you build a **reusable context** for each one, and starts new
Claude sessions already primed with it — so you stop re-explaining your project every
time. Sessions keep running even after you close the app, and you can schedule
read-only runs to check in on a project while you're away.
