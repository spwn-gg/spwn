title: Building from Source
description: spwn is open source — build it from source or contribute.
---

Just want to **use** spwn? Head to the [Installation guide](/spwn/getting-started/installation/) —
you don't need to build anything.

spwn is open source. If you want to build it from source or contribute, the full
instructions — native macOS build, the Docker development workflow, and how to build
these docs — live in the repository:

- **[BUILD.md](https://github.com/spwn-gg/spwn/blob/main/BUILD.md)** — build and
  development guide.
- **[github.com/spwn-gg/spwn](https://github.com/spwn-gg/spwn)** — source, issues,
  and releases.
